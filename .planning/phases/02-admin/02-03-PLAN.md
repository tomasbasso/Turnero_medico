    GET handler — authenticated (any valid role can read):

    - Signature: export async function POST(request: NextRequest)
    - Auth guard: const authResult = requireAdmin(request); if (authResult instanceof Response) return authResult

    - Create: const specialty = await prisma.specialty.create({ data: { name: name.trim(), description: description ?? null, color: color ?? '#0d9488' } })
    - Return: Response.json({ specialty }, { status: 201 })

    NEVER use NextResponse.json(). Always use Response.json().
    NEVER spread body directly into Prisma data objects.
